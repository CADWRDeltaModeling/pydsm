pydsm.analysis.oprule_parser.parse_action(text: str) → List[ActionSpec][source]’āü

Parse a DSM2 OPERATING_RULE ACTION string into a list of ActionSpec.

Handles WHILE/THEN-chained compound actions (both treated as simultaneous). Any RAMP N MIN suffix is parsed and stored in ActionSpec.ramp_min but ramp-interpolation is not applied.
