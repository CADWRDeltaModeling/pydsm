pydsm.output.ptm module

This module contains functions for Particle Tracking Model

pydsm.output.ptm.create_frame_for_anim_data(ptm_data)[source]

Creates a pandas dataframe from the numpy array data structure The data frame is multi indexed with an index of datetime and particle id and the columns have the channel id , x,y,z position and the value field

pydsm.output.ptm.load_anim_data(ptm_file)[source]

Reads the PTM Animation binary file format. This is Java binary output file and here numpy is used to load all the records into a numpy structure and returns that

pydsm.output.ptm.load_ptm_trace(trace_file)[source]

Loads the trace file output from PTM as a pandas dataframe and associate meta data as a dictionary returns the data frame and a dictionary containing the start_date, end_date, timestep and number of particles

Example Usage: df,meta_info=load_ptm_trace(‘./dsm2_v8.2.0b1/studies/historical/output/trace.out’)
